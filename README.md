# Coinone
[![GemVersion](https://badge.fury.io/rb/coinone.svg)](https://badge.fury.io/rb/coinone)

`Coinone` Gem is porting from Coinone API


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'coinone'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install coinone
